# File-Based Library Management System


# Description

A **File-Based Library Management System** built using Java.  
All book records are stored in a `.dat` file so data **persists** even after the program is closed.

---

##  Features

```
1.  Add New Book
2.  View All Books
3.  View Book by ID
4.  Search by Title
5.  Search by Author
6.  Issue a Book
7.  Return a Book
8.  View Issued Books
9.  Delete a Book
10. Export Library Report (.txt)
11. Exit
```

##  Java Concepts Used

 File Handling `ObjectOutputStream`, `ObjectInputStream`, `PrintWriter` 
 Exception Handling `try-catch` for file errors and invalid input
